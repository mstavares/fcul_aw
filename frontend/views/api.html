Api

Disease Service

  • GET top N related articles to a disease

    Usage: disease/get_top_articles?lim={lim}&diseaseId={diseaseId}

  • GET first N diseases

    Usage: disease/get_all/{limit}

  • GET diseases given a fragment of the name

    Usage: disease/get_fragment/{fragment}

  • GET full disease

    Usage: disease/get/{id}

  • GET Top N related diseases

    Usage: disease/get_top_related_diseases?lim={lim}&diseaseId={diseaseId}

  • GET statistics

    Usage: disease/get_statistics

Pubmed Service

  • GET diseases mentioned in an article and where they appear in text

    Usage: pubmed/get_mentioned_diseases/{pubmedId}

Feedback Service

  • Change feedback on pubmed

    Usage: feedback/pubmed?diseaseId={id}&pubmedId={pId}&operation={op}

  • Change feedback on tweets

    Usage: feedback/tweet?diseaseId={id}&tweetId={tId}&operation={op}

  • Change feedback on images

    Usage: feedback/image?diseaseId={id}&imageId={iId}&operation={op}

op = 0 => increment implicit feedback

op = 1 => increment explicit feedback

op = 2 => decrement implicit feedback

op = 3 => decremente explicit feedback
